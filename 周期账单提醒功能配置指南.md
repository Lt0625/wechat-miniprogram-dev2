# 周期账单提醒功能配置指南

## 功能说明

周期账单提醒功能会在以下情况下工作：

1. **应用启动时**：检测今日到期的周期账单，弹出提醒对话框
2. **开启提醒**：用户开启某个周期账单的提醒开关时，请求微信订阅消息权限
3. **云函数发送**：通过云开发发送订阅消息到用户微信

## 配置步骤

### 第一步：获取订阅消息模板

1. 登录 [微信小程序后台](https://mp.weixin.qq.com)
2. 进入 **功能** → **订阅消息**
3. 点击 **添加模板**
4. 选择合适的模板，建议关键词：
   - 账单名称（phrase1）
   - 到期日期（date2）
   - 金额（amount3）
   - 备注（remark4）
5. 复制获得的 **模板 ID**

### 第二步：替换模板 ID

修改以下两个文件中的 `YOUR_TEMPLATE_ID`：

1. `cloudfunctions/sendReminder/index.js`
2. `miniprogram/pages/recurring/recurring.js`

```javascript
const templateId = '你的模板ID'  // 例如：'abc123xxxxxxx'
```

### 第三步：部署云函数

1. 在微信开发者工具中打开项目
2. 找到 `cloudfunctions/sendReminder/` 文件夹
3. **右键点击** → **上传并部署**
4. 等待部署完成

### 第四步：配置云开发环境（如果尚未配置）

1. 在 `miniprogram/app.js` 中已配置云开发环境：
   ```javascript
   initCloud() {
     wx.cloud.init({
       env: 'cloudbase-d1g8mjg3m02a6eba2',  // 使用语音记账的云环境
       traceUser: true
     })
   }
   ```

## 使用流程

1. 用户打开小程序
2. 如果有今日到期的周期账单，弹出提醒对话框
3. 用户点击"去记账"跳转到首页
4. 用户可以点击周期账单列表中的"添加"按钮快速生成账单

## 注意事项

1. **订阅消息限制**：微信对订阅消息有次数限制，用户需要主动授权
2. **模板匹配**：订阅消息的字段需要与模板关键词匹配
3. **本地提醒**：即使不发送订阅消息，应用启动时的弹窗提醒依然有效

## 故障排查

### 订阅消息发送失败
- 检查云函数是否部署成功
- 检查模板 ID 是否正确
- 检查用户是否授权订阅

### 弹窗不显示
- 确认周期账单的 `nextDue` 日期是否正确
- 确认 `enabled` 和 `reminderEnabled` 都为 `true`
