## 第四步：开启云开发的语音识别能力

云函数中使用了两种识别方案，任选其一：

### 方案 A：微信云开发 AI 能力（推荐，免费）

1. 进入微信云开发控制台 → **扩展能力** → **AI 能力**
2. 开启**语音识别**能力
3. 云函数中的 `cloud.ai.run` 会自动生效

### 方案 B：腾讯云 ASR（更强大，有免费额度）

1. 前往[腾讯云控制台](https://console.cloud.tencent.com/)
2. 开通**语音识别**服务（每月1万次免费）
3. 创建 API 密钥（SecretId + SecretKey）
4. 在云函数中配置（修改 `cloudfunctions/voiceBill/index.js`）：

```javascript
// 在云函数顶部添加腾讯云 SDK
const tencentcloud = require('tencentcloud-sdk-nodejs')
const AsrClient = tencentcloud.asr.v20190614.Client

const client = new AsrClient({
  credential: {
    secretId: 'your-secret-id',
    secretKey: 'your-secret-key'
  },
  region: 'ap-shanghai',
  profile: {}
})
```

## 常见问题

**Q：提示"云开发初始化失败"？**  
A：检查 `add-bill.js` 中的 `env` 是否替换为正确的环境 ID

**Q：云函数调用超时？**  
A：默认超时 3 秒，可在云开发控制台将 `voiceBill` 云函数超时时间改为 10 秒

**Q：识别准确率低？**  
A：目前使用关键词匹配，说话时尽量包含关键词（如"花了"、"元"、具体分类名）

**Q：微信开发者工具录音测试失败？**  
A：开发者工具不支持真实录音，需要在真机上测试语音功能
